# LampExample

## Install Requirements
Update packages
```sh
sudo apt update
```
Install apache2 package
```sh
sudo apt install apache2 -y
```
Install php
```sh
sudo apt install php libapache2-mod-php -y
```
## Setup
Navigate to apache2 web directory
```sh
cd /var/www/html
```
Delete index.html default file
```sh
sudo rm index.html
```
